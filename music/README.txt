HƯỚNG DẪN THÊM NHẠC NỀN
========================

1. Đặt file nhạc vào thư mục này với tên: background-music.mp3
   Hoặc background-music.ogg

2. Các định dạng được hỗ trợ:
   - MP3 (.mp3) - Khuyến nghị
   - OGG (.ogg) - Dự phòng

3. Tên file phải chính xác:
   - background-music.mp3
   - background-music.ogg

4. Đề xuất:
   - File nhạc nên có chất lượng tốt (128kbps trở lên)
   - Thời lượng phù hợp (2-5 phút, sẽ tự động lặp lại)
   - Nhạc không lời hoặc nhạc nền nhẹ nhàng

5. Sau khi thêm file nhạc:
   - Mở lại trang web
   - Nhấn vào nút nhạc ở góc dưới bên trái để bật/tắt nhạc
   - Nhạc sẽ tự động lặp lại khi phát

Lưu ý: Nếu không có file nhạc, nút nhạc vẫn sẽ hiển thị nhưng không phát được.
